## apexfmt

Format Apex

```
apexfmt [file...]
```

### Options

```
  -h, --help      help for apexfmt
  -l, --list      list files whose formatting differs from apexfmt's
  -s, --soql      format SOQL query
  -v, --verbose   enable debug logging
  -w, --write     write result to (source) file instead of stdout
```
